# Диплинки

Для обеспечения backend-driven подхода можно исопльзовать диплинки
Поддержанные сейчас диплинки:

- `deeplink://dormitory/parent_agreement/upload_file?applicationId={id}` - загрузка согласия родителей для общежития
- `deeplink://common/download_file?fileUrl={url}` - скачивание файла по URL (например, PDF приказа)
